# Visual Time

My very first dabble with a programming project was this: a *pomodoro timer*.

<br />

After learning the benefits of timeboxing and taking regular breaks after certain period of focus, I decided that I would like something that is **pleasing to look at**, **ergonomic** and **easy to use**.

<br />

At this point, I was already learning the basics of programming and following YouTube tutorials. I used a library called [***p5.js***](https://p5js.org) which I found from the sweet channel called [The Coding Train](https://www.youtube.com/@TheCodingTrain). It has a runtime loop with which drawing animations can be done.

<br />

I used this, and a little bit of math to figure out the gradually disappearing "pie-chart", according to the current time based on the total selected amount.
